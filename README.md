# vue-stack-2.0
Vue 2.0 project boilerplate.

### Includes
* Vue 2.0
* Vue Router 2.0
* Vue Resource
* Vue Stash 2.0
* jQuery
* Boostrap Sass

### Setup
* Clone the git repo `git clone https://github.com/cklmerver/vue-stack-2.0`
* Change into the directory `cd vue-stack-2.0`
* Run `npm install`
* Run `gulp`
